React
=====

## Covered

- Create-React-App v2: TypeScript & SASS
- Fragments: Avoid superfluous `<div>`s
- Suspense & lazy: Code-Splitting
- Error Boundaries: Recover from render() crashes (not event handlers)
- Context: Global variables
- Memo
- Scheduler
- de nieuwe profiler

cd react-socks
